## Sobre

Repositório com os materiais de estudo da formação **Data Science Aplicado à Saúde**, realizada entre 2024 e 2025 pela **Universidade Federal de Roraima (UFRR)**, executada pelo **Departamento de Ciência da Computação (DCC)** no laboratório **Fox Data / CIT**.

O programa teve carga horária total de **236 horas**, distribuídas em 9 módulos — 8 disciplinas técnicas mais um módulo prático de encerramento (Hands On) — cobrindo desde fundamentos de análise de dados em Python até construção de dashboards aplicados à área da saúde.

## Stack

- **Linguagem:** Python 3.x
- **Análise de dados:** pandas · numpy
- **Visualização:** matplotlib · seaborn
- **Machine Learning:** scikit-learn
- **Dashboards:** streamlit
- **Ambiente:** Jupyter Notebook · Google Colab

## Como executar os notebooks

```bash
# Clonar o repositório
git clone https://github.com/DilliKel/data-science-saude-ufrr.git
cd data-science-saude-ufrr

# Instalar dependências
pip install pandas numpy matplotlib scikit-learn streamlit jupyter

# Abrir o Jupyter
jupyter notebook
```

> Todos os notebooks foram desenvolvidos e testados no **Google Colab**. Para rodar sem instalação local, abra o arquivo `.ipynb` diretamente no Colab.

## Aprendizados

Esta formação foi minha entrada formal em **Data Science** — área que complementa minha trajetória de desenvolvimento frontend com capacidade analítica e de visualização de dados. Os principais aprendizados foram:

- Manipulação e limpeza de dados reais da área da saúde com pandas
- Construção de visualizações informativas para públicos não técnicos
- Análise espacial aplicada à vigilância epidemiológica
- Desenvolvimento de dashboards interativos com Streamlit
- Aplicação de modelos de ML em contextos de saúde pública
