## Sobre

Este repositório reúne os materiais de estudo da formação **Data Science Aplicado à Saúde**, realizada entre 2024 e 2025 pela **Universidade Federal de Roraima (UFRR)**, executada pelo **Departamento de Ciência da Computação (DCC)** no laboratório **Fox Data / CIT**.

O programa teve carga horária total de **236 horas**, distribuídas em 9 módulos — 8 disciplinas técnicas mais um módulo prático de encerramento (Hands On) — cobrindo desde fundamentos de análise de dados em Python até a construção de dashboards e aplicações voltadas à área da saúde.
